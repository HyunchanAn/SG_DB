# 데이터 정규화(Terminology Normalization) 전수조사 및 딕셔너리 구축안

본 문서는 현장 연구원들이 엑셀과 Access DB에 수기로 작성한 텍스트 피처(Feature)들의 변칙 사례들을 전수 조사하여, 향후 AI 학습용으로 하나의 "표준어"로 묶기 위한(Mapping) 제안서입니다. 
확인해 보시고, 제안된 **[표준 통일안]**에 대한 의견이나 수정 사항을 알려주시면 최종 딕셔너리 코드를 `load_data.py`에 이식하겠습니다.

---

## 1. 실험 현상 및 결함 (비고란)
동일한 화학적 현상이나 불량을 작업자마다 각기 다른 표현으로 기재했습니다.

| 분류 | 실제 기입된 원본 텍스트 (Variations) | [표준 통일안 제안] |
|:---:|:---|:---|
| **점도 이상** | `겔레이션 도포 불가`, `점도 상승으로 인한 도포불가`, `이형제 배합 48시간 후 굳음`, `점도상승` | ➡️ **`[점도상승_겔화]`** |
| **결과 불량** | `~보다 안좋음`, `NG`, `도포 미진행(점착제 자체 NG)`, `미비`, `불량함` | ➡️ **`[성능미달_NG]`** |
| **미세 오염** | `미세백탁`, `미세오염`, `그림자`, `미세얼룩` | ➡️ **`[표면미세오염]`** |
| **석출 현상** | `석출발생`, `석출`, `결정석출` | ➡️ **`[성분석출]`** |

## 2. 테스트 조건 및 결과 텍스트 (내열성, 내후성 등)
기존 정규 포맷 `(결과)*(피착재/조건)` 규칙을 위반한 띄어쓰기 및 순서 역전 사례입니다.

| 분류 | 실제 기입된 원본 텍스트 (Variations) | [표준 통일안 제안] |
|:---:|:---|:---|
| **띄어쓰기** | `(7일양호)*(BA)`, `(7일 양호)*(BA)`, `(12일 양호)*(BA)` <br> `(43일박리불가)(BA)`, `(43일 박리불가)*(BA)` | ➡️ 정규표현식으로 내부 공백 모두 제거 및 누락된 `*` 일괄 복구 |
| **포맷 역전** | `(탈리)*(BA/150도15분)` (내열성) | ➡️ `(탈리)*(150도15분/BA)` 로 순서 교정 |
| **컬럼 오기입** | `(10일양호)*(BA)` (내열성 컬럼에 기입됨) | ➡️ 내후성 컬럼으로 데이터 이동 (옵션) 또는 유지 |
| **다중 기입** | `(양호)*(150도15분/BA)+(양호)*(60도3d/AL프로파일)` | ➡️ 쪼개서 멀티 라벨(Multi-label) 배열 처리 제안 |

## 3. 피착재(기재) 명칭 동의어
표면 특성이 거의 동일한 피착재를 여러 이름으로 부르고 있습니다.

| 분류 | 실제 기입된 원본 텍스트 (Variations) | [표준 통일안 제안] |
|:---:|:---|:---|
| **스테인리스** | `BA` (Bright Annealed), `SM` (Stainless Mirror) | ➡️ **`BA_SM`** (하나의 범주로 병합) |
| **알루미늄** | `AL프로파일`, `유광AL프로파일` | ➡️ **`AL_Profile`** |
| **우드패턴** | `WPVC`, `WPO` | ➡️ **`Wood_Film`** (두 재질의 표면장력 차이가 크다면 분리 유지 필요) |
| **언어차이** | `투명`, `Clear` | ➡️ **`Clear`** |

## 4. 공동 작업자 명칭
`박다영, 진호철`, `진호철, 박다영`, `노항우, 진호철, 정주현` 등 콤마(,)로 여러 명을 기재했으나 순서가 뒤죽박죽인 경우가 약 50여 건 있습니다. 
➡️ **[제안]:** 이름 텍스트를 Split 한 뒤 **가나다 순(알파벳 순)으로 정렬**하여 다시 결합하면 100% 동일한 해시(Hash)값으로 AI가 인식합니다.

---

### 💬 검토 요청 사항
위 제안된 통일안(Mapping Dictionary) 중 **"이건 합치면 안 되고 별도의 화학적 피처로 분리해야 해!"**라고 생각하시는 부분이 있으신가요? (예: `BA`와 `SM`은 완전히 다른 조건으로 봐야 한다 등) 

의견을 남겨주시면, 피드백을 반영하여 `load_data.py` 내부의 AI 데이터 정제용 전처리 함수(`TextNormalizer`)를 최종 완성하겠습니다!
